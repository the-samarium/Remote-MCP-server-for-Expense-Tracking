# Expense Tracker MCP

A simple Python-based expense tracker created as part of the Model Context Protocol (MCP) learning project. It helps users record, categorize, and review expenses while demonstrating a clean, small-scale Python application.

## What it does

- Tracks expense entries with amount, date, and category
- Summarizes total spending and transaction history
- Provides a lightweight foundation for learning MCP app structure

## Why it exists

This project is designed for practice with creating an MCP application design and MCP concepts. It is intentionally compact, easy to extend, and useful for learning how to manage data in a simple CLI-style tool.

## Getting started

1. Activate your Python environment
2. Run the app with `python main.py`
3. Add or review expenses as needed

## Extend it

You can improve this project by adding:

- category reports and charts
- input validation and command-line options
- a graphical or web-based interface
